Contextual Classes

Contextual classes can be used to color the table, table rows or table cells. The classes can be used are: .tabl-primary, .table-secondary, .table-sucess, .table-info, .table-warning, .table-danger, .table-active, .table-secondary, .table-light and .table-dark:
